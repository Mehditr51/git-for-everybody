# 🌿 What is Git?

Git is the world's most popular version control system.

If you've ever saved a file as:

- project-final.docx
- project-final-v2.docx
- project-final-final.docx

...Git solves that problem in a much smarter way.

Instead of creating dozens of copies of your files, Git keeps track of every change you make over time.

# 📖 What is Version Control?

Version Control is software that records changes made to files over time.

It allows you to:

- Track changes
- Restore old versions
- Compare versions
- Undo mistakes
- Work with other people safely

Think of it like **Save Points** in a video game.

Every important moment can be saved.

If something goes wrong...

You simply go back.

# 📝 Real-Life Example

Imagine you're writing an essay.

Instead of:

```

```

Git keeps every version automatically.

You can compare them anytime.

# 🧠 Git vs GitHub

Many beginners confuse these two.

| Git | GitHub |
|------|---------|
| Software installed on your computer | Website |
| Tracks file history | Stores Git projects online |
| Works without internet | Requires internet |
| Free and open-source | Cloud platform built around Git |

A simple way to remember:

> Git is the engine.
>
> GitHub is the garage.

---

# 📚 A Tiny Bit of History

Git was created in **2005** by **Linus Torvalds**, the creator of Linux.

He built Git because existing version control tools didn't meet the needs of the Linux project.

Today, Git is the industry standard.
