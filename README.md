# 🌿 Git for Everybody

A beginner-friendly guide to Git — written for people with **zero coding background**.

No jargon. No assumptions. Just clear explanations, real examples, and visuals.

---

## 🤔 Why This Repo Exists

Most Git tutorials assume you already know what a terminal is.
This one doesn't.

If you've ever thought *"I have no idea what commit, branch, or push means"* — this is for you.

## 💡 Key Concepts at a Glance

| Term | What it means in plain English |
|------|-------------------------------|
| **Repository (repo)** | A folder that Git is tracking |
| **Commit** | A saved snapshot of your files |
| **Branch** | A parallel version of your project |
| **Merge** | Combining two branches together |
| **Push** | Uploading your commits to GitHub |
| **Pull** | Downloading changes from GitHub |
| **Clone** | Copying a repo from GitHub to your computer |
